# Игра в слова (Wordle Clone)

«Игра в слова» — это мобильное Android-приложение/учебный проект, представляющее собой аналог популярной головоломки Wordle. Игроку необходимо отгадать загаданное слово из 5 букв за 6 попыток. Приложение полностью на русском языке и включает встроенный словарь для проверки вводимых слов.

## Особенности проекта

* **Классическая механика Wordle:** У вас есть 6 попыток, чтобы угадать слово из 5 букв.
* **Цветовая индикация:**
  * 🟩 **Зеленый:** Буква есть в слове и находится на своем правильном месте.
  * 🟨 **Желтый:** Буква есть в слове, но находится на другом месте.
  * ⬜ **Серый:** Такой буквы в загаданном слове нет.
* **Интерактивная экранная клавиатура:** Клавиатура также окрашивается в соответствующие цвета по мере отгадывания букв, помогая быстрее находить правильный ответ.
* **Проверка по словарю:** Введенное слово проверяется по встроенной базе. Приложение не позволит ввести несуществующее слово или случайный набор букв.
* **Система уровней:** После успешного (или неуспешного) завершения раунда можно перейти на следующий уровень с новым словом.
* **Оффлайн работа:** Словарь загружается локально из ресурсов приложения (`res/raw/words`), поэтому для игры не требуется подключение к интернету.

## Стек технологий

* **Язык программирования:** Kotlin (версия 2.0.21).
* **Пользовательский интерфейс:** Jetpack Compose с использованием компонентов Material Design 3.
* **Архитектура:** Использование `ViewModel` для управления состояниями игры (состояние сетки букв, статус клавиатуры, текущий уровень).
* **Android SDK:** Минимальная версия — API 24 (Android 7.0), целевая версия — API 35.

## Как запустить проект

1. **Клонирование:** Склонируйте репозиторий на свой компьютер.
2. **Среда разработки:** Откройте проект в Android Studio (рекомендуется версия с поддержкой Android Gradle Plugin 8.9.0).
3. **Словарь:** Убедитесь, что в папке `app/src/main/res/raw/` присутствует файл `words` со списком слов.
4. **Сборка и запуск:** Нажмите `Shift + F10` (или кнопку Run в Android Studio), чтобы скомпилировать приложение и запустить его на эмуляторе или физическом устройстве.
